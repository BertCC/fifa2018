This is a fifa 2018 betting game developped using meteor/react/bootstrap. It is based on https://github.com/lu22do/accounts-boilerplate-react.

The UI react components are the following:
- App
- Topbar
- Auth:
    - Login
    - Register
- Users: 
    - User list
- All teams  
- Your teams:
    - Team list
    - New team
    - Edit team
